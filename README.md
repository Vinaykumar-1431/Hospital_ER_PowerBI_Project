# Hospital ER Analysis Dashboard | Power BI Project

## 📌 Project Overview

This project focuses on analyzing **Hospital Emergency Room (ER) data using Power BI** to understand patient visits, waiting time, admission trends, and healthcare operational performance.

The objective of this project is to transform raw hospital data into an interactive dashboard that helps stakeholders monitor ER efficiency and make data-driven decisions.

## 🛠️ Tools & Technologies

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization
* Excel / CSV Dataset

## 📂 Dataset Overview

The dataset contains Emergency Room patient records including:

* Patient details
* Visit information
* Admission status
* Waiting time
* Department details
* Patient satisfaction metrics

## 🔄 Data Preparation Process

Performed data cleaning and transformation using **Power Query**:

* Removed duplicates
* Handled missing values
* Changed data types
* Created calculated columns
* Prepared data for reporting

## 🏗️ Data Modeling

Created relationships between tables and built a structured data model for efficient analysis.

Used:

* Relationships
* Calculated columns
* Measures
* DAX calculations

## 📊 Dashboard Analysis

The Power BI dashboard provides insights on:

### Patient Analysis

* Total ER visits
* Patient distribution
* Age group analysis
* Gender-wise analysis

### Operational Analysis

* Average waiting time
* Patient arrival trends
* Department performance
* Admission vs discharge analysis

### Performance Metrics

Created KPIs for:

* Total Patients
* Average Wait Time
* Satisfaction Score
* Admission Rate
